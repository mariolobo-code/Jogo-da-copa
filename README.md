O usuário informa os placares das partidas entre Brasil, Marrocos, Haiti e Escócia.
O sistema processa os resultados inseridos.
Os pontos são distribuídos automaticamente de acordo com o resultado de cada jogo.
São calculados os gols marcados, gols sofridos e o saldo de gols de cada seleção.
A tabela é atualizada e exibida em ordem de classificação.
O primeiro colocado é destacado em verde e o último colocado em vermelho para facilitar a visualização.
🎯 Objetivo

Permitir a simulação dos resultados dos jogos do Grupo C da Copa do Mundo 2026 e apresentar automaticamente a classificação das seleções com base na pontuação, gols marcados, gols sofridos e saldo de gols.

A proposta do sistema é permitir que o usuário insira os resultados das partidas entre as equipes participantes e visualize automaticamente a tabela de classificação atualizada. O programa calcula a pontuação de cada seleção, os gols marcados, os gols sofridos e o saldo de gols, organizando os times conforme seu desempenho.

Além de proporcionar uma experiência relacionada ao futebol, o projeto busca aplicar os conhecimentos adquiridos na disciplina de Introdução à Programação, utilizando conceitos como variáveis, listas, dicionários, funções, estruturas condicionais, estruturas de repetição e interface gráfica com Tkinter.

O sistema foi desenvolvido para ser simples, intuitivo e visualmente agradável, permitindo que qualquer usuário acompanhe facilmente a evolução da classificação das equipes.#
